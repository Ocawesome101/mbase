NAME
        dmesg - view system logs or events

SYNOPSIS
        dmesg
        dmesg [-e]

DESCRIPTION
        WARNING: DO NOT USE DMESG IN OCVM, EXCEPT WITH THE -e COMMAND-LINE FL
AG.  IT DOES NOT WORK.

        Prints system logs.  If the -e flag is specified, behaves similarly to
 OpenOS's dmesg, and prints all recieved events to the console.
