NAME
        ed - the standard Unix text editor

SYNOPSIS
        ed [file]

DESCRIPTION
        ed is a Lua implementation of the standard Unix text editor.  It does n
ot feature all of the commands, but is similar enough to be recognizable.  It su
pports the a, c, d, i, l, p, P, q, s, and w commands.

EXAMPLES
        Command parameters should be given without spaces - for example, w exam
ple becomes wexample.

        1,5s/old/new/
            Substitute old for new on lines 1 through 5.

        ,p
            Print the whole buffer.
