NAME
        free - display amount of total, free, and used memory in the system

SYNOPSIS
        free [-h]

DESCRIPTION
        free displays the total amount of free and used physical memory in the 
system.  The displayed rows are:

        total
            Total installed memory

        used
            Used memory (calculated as total - free)

        free
            Unused memory
