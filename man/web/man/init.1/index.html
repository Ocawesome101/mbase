NAME
        InitMe, init - Monolith userspace initialization

DESCRIPTION
        InitMe is Monolith(5)'s default init system.  It initializes the i
o, package, and initsvc libraries and cleans up the global environment.  Fo
r more detail on what APIs are provided to the init process, see init(5).
