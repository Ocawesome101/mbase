        list(): table
                Returns a table of services, and whether they are running.  Each
 entry is a sub-table with the fields name:string and running:boolean.
        
        stop(service:string): boolean or nil, string
                Attempts to stop service service.  May have compatibility with
 OpenOS services.

        enable(service:string): boolean or nil, string
                Attempts to enable service service.

        disable(service:string)
                Disable service script.
