NAME
        io - reimplementation of the Lua standard io library

DESCRIPTION
        io is the Monolith re-implementation of the Lua-standard library of the
 same name.  It is intended to function identically.

        For documentation, see https://lua.org/manual/5.3/manual.html#6.8.
