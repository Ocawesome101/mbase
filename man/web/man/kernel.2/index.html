NAME
        kernel - low-level system interfaces

DESCRIPTION
        kernel is a global table provided to the init process by the kernel.  I
t contains such low-level kernel functions as thread and user management.  The 
kernel API provides five sub-fields: the scheduler, kernel.thread(2); files
ystem management through kernel.filesystem(2); kernel.logger(2), a basic
 bootlogger; user management through kernel.users(2); and module management 
in kernel.module(2).

NOTES
        kernel.module is untested.
