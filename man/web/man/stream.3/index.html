NAME
        stream - create unbuffered faux io streams

DESCRIPTION
        stream is an API provided by Monolith, used in the io(3) library fo
r standard input and output streams.

METHODS
        new(read:function, write:function, close:function[, fields
:table]): table
                Creates a stream with the methods read, write, and close a
ssigned to their corresponding arguments.  Any additional fields specified in th
e fields table are also copied to the output stream.
